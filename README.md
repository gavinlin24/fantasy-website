# Fantasy Insights

**Live Demo:** [https://fantasyinsights.up.railway.app](https://fantasyinsights.up.railway.app)

## Overview

**Fantasy Insights** is a full-stack web application that delivers real-time fantasy sports analytics, AI-driven insights, and league activity tracking through a modern user interface.

**Tech Stack:**  
FastAPI • React (Vite) • Tailwind CSS • Google Gemini API • ESPN Fantasy API • Railway

## Current Features

- **AI-powered insights** — Integrates Google Gemini to generate natural-language recommendations such as trade advice and lineup optimization.  
- **Real-time ESPN integration** — Fetches and normalizes live fantasy league data for consistent presentation across the frontend.  
- **Smart caching** — Backend memoization reduces redundant API calls and optimizes Gemini token usage.  
- **Secure deployment** — Fully HTTPS-enabled on Railway with isolated environments and continuous deployment.
